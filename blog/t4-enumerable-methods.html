Enumerables

The Map method

May 3, 2015


The Enumerable module in Ruby is probably the reason a lot of Ruby developers don't rely on the for loop much. I'm pretty accustomed to using the for loop, and this is why I still get thrown for a loop (sorry) with enumerables. But I'm learning to love them! They provide really convenient ways to traverse, search, sort, and manipulate collections. Here, I'm going to look at the map method. The map method executes a given block for each element of an array or hash, and will actually return a new array. It will not, however, manipulate the original array unless you use map!, which indicates it's destructible (we won't get into that for now). I'll demonstrate two use examples.
